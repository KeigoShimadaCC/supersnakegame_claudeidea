# Supersnake Game (React + TypeScript + Vite)

This is a React + TypeScript game project scaffolded with Vite.

## Security / secrets

- Do **not** commit any real API keys, tokens, passwords, or private keys.
- Keep environment variables in local `.env*` files (these are ignored by git via `.gitignore`).
- If you ever accidentally commit a secret, rotate it immediately and use `git filter-repo` or the GitHub UI to remove it from history.
